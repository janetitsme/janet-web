Android Mobile Application

Using Java

The purpose of the app is to query a server which contains information about the food hygiene ratings from the Food Standards Agency for all restaurants in the UK. The user can search in three different ways, by GPS coordinates, by business name and by postcode. The server returns data about the closest 10 businesses in JSON format. The app then presents the data in table format, using relevant pictures for the ratings. By clicking on one of the names, the app loads a Google Map showing the locations of all the results of the search, with the one you clicked on highlighted and centred.

  • Category: Mobile Development on Android

Data Engineering - Machine Learning

The aim of the assignment was to analyse this bank loan dataset and see whether the outcome prediction could be improved whether the customer default the loan. This analysis took a number of forms.

The dataset for this assignment was a bank loan dataset that contained personal information about 2000 bank customers, such as their name and address, age, gender, how long they have lived at their address, their employment status, any current debt, whether they own their home, if they have any CCJs against them, how much they wanted to borrow and whether or not they defaulted on the loan.

Data Summary First step was to list the data in the dataset, for each attribute describe whether it is nominal or numeric, continuous or discrete, and whether or not it would be useful in building the solution.

Data Mining Algorithm Selection Select and describe two different types of data mining algorithms that would be appropriate for solving the problem. The description included the theoretical algorithm and its core features, including methodology, and pros and cons.

Data Preparation Prepare the data by identifying and dealing with noise and outliers, in order that a cleaner dataset would give more reliable results.

Modelling Results and Discussion In this section the dataset is run through a series of experiments, using different testing strategies such as 50:50 and 10 fold cross validation. The parameters for the different strategies are changed one at a time, with the best result being carried forward.

The program used to analyse the data was Weka 3: Data Mining Software in Java

The final report for this assignment can be found here: Assignment Report

The dataset used for the assignment was the Loan Dataset
